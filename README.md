# Kyle /_ — Creative Portfolio

Personal portfolio website for Kyle — automotive photographer, video editor, and web developer based in Brighton & Hove.

## Stack

- Vanilla HTML5, CSS3, JavaScript (no frameworks, no dependencies)
- Hosted on GitHub Pages / Vercel
- Google Fonts: Bebas Neue, Share Tech Mono, Rajdhani

## Structure

```
/
├── index.html              ← Homepage
├── style.css               ← Global styles
├── main.js                 ← Canvas background, scroll reveal, interactions
├── README.md
└── docs/
    ├── docs.css            ← Shared documentation page styles
    ├── landscape-video.html
    ├── portrait-video.html
    ├── landscape-photos.html
    └── portrait-photos.html
```

## Features

- Animated `+` sign canvas background that reacts to mouse position
- Neon cyan/blue theme with flickering neon text effects
- Scroll-triggered content reveal
- Four portfolio placeholders: landscape video, portrait video, landscape photo carousel, portrait photo carousel
- Each portfolio piece links to a dedicated documentation page
- Three certification placeholders
- About Me section placeholder
- Cursor glow trail effect
- Fully responsive
